Summary:
Dnsquery is a tool that searches the given domain for basic DNS information such as A, NS, MX, SOA etc.
Additionally, it searches for ASN records, registrar and the expiration date of the domain.
It also evaluates whether multiple DNS are nested in the same IP and check for differences between Whois records and DNS records.
Lastly, it shows the Email service provider of the given domain( if there is one ), if the domain uses "Office 365 services", it will then check whether the required records such as SPF, SRV are present.
